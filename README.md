# cluedo-app

- firebaseのREALTIME DATABASEを活用したアプリケーションです。

## firebase起動
`$ npm run serve-hosting `

## 環境設定

- [デプロイ](https://oga-game.web.app/)
  - EC2
  - RDB
  - firebase realtime database

- 仮想開発環境
  - vagrant
  - firebase realtime database

- ローカル開発環境
  - firebase local host
  - firebase realtime database

## 開発環境

- バックエンド
  - JavaScript
  - PHP(予定)

- フロント
 - HTML
 - CSS
 - SCSS
 - JavaScript

- 対応ブラウザ
  - Google Chrome
